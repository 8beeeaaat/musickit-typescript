# musickit-typescript

TypeScript type definitions for [MusicKit JS](https://developer.apple.com/documentation/musickitjs).

## Installation

```sh
# using npm
npm install --save-dev musickit-typescript

# using yarn
yarn add --dev musickit-typescript
```
